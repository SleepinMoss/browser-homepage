# browser-homepage
Custom homepage for your browser

*Installation*
1. Download zip file
2. Extract zip file
3. Open your browser settings and set your homepage to: file:///path/to/browser-homepage/main.html

*Customization*
If you understand css and html (or just ask AI to edit it), you can easily customize the homepage by editing .html and .css file.
